# brute-force-simulation-medusa

# Projeto:

Este repositório documenta minha jornada prática utilizando **Kali Linux**, **Medusa** e ambientes vulneráveis (como **Metasploitable 2** e **DVWA**) para entender ataques de força bruta, suas consequências e formas de mitigação.

## Objetivos do Projeto

- Compreender como ataques de força bruta funcionam na prática.
- Testar o Medusa em diferentes serviços: FTP, DVWA e SMB.
- Documentar comandos, passos, resultados e prints.
- Analisar vulnerabilidades comuns.
- Propor medidas reais de defesa e mitigação.
- Utilizar o GitHub como forma de registro e portfólio.

---

## Cenários Planejados

1. **Força bruta em FTP** usando Medusa  
2. **Automação de login no DVWA** (medium / low security)
3. **Password spraying em SMB** com enumeração de usuários
4. **Análise e mitigação**

As evidências serão adicionadas conforme avanço no desafio.

---

## Ambiente Utilizado

- **Host:** Ubuntu Linux  
- **VirtualBox**
- **Máquina atacante:** Kali Linux  
- **Máquina alvo:** Metasploitable 2  
- **Ferramentas principais:**
  - Medusa  
  - Nmap  
  - DVWA
